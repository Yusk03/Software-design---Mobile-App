## 🎉 Get started

**1.** Clone project
```bash
git clone https://github.com/Yusk03/Software-design---Mobile-App.git
```
**2.** download all dependencies
```bash
$ yarn
```
**3.** Build app and run it
```bash
$ yarn android
```
**4.** Start bundling
```bash
$ yarn start
```

#### *or use Android Studio*.
